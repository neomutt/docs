# Start Here

You do not need to know anything about terminal email clients to get started. This page will ask a few quick questions and point you to the right steps.

## 1. Where is your email hosted?

Pick the option that matches your account:

- Gmail: go to [Setting Up NeoMutt with Gmail](/home/mutt/rtd/docs/docs/tutorials/gmail-setup.md)
- Microsoft 365 / Outlook: go to [Setting Up NeoMutt with Office 365](/home/mutt/rtd/docs/docs/tutorials/office365-setup.md)
- Other provider or self-hosted IMAP: go to [How to Set Up IMAP](/home/mutt/rtd/docs/docs/howto/imap.md)

If you are not sure, open your current mail client and check the server name. It will usually include `imap.` or `mail.`.

## 2. Do you already have mail on this machine?

- Yes, I already have mail stored locally: go to [Mailbox Formats](/home/mutt/rtd/docs/docs/howto/mailbox-formats.md)
- No, I want to connect to a server: continue to [Writing Your First Configuration](/home/mutt/rtd/docs/docs/tutorials/first-config.md)

## 3. Do you want the simplest possible setup?

- Yes, just get it working: use [Writing Your First Configuration](/home/mutt/rtd/docs/docs/tutorials/first-config.md)
- I want help with a specific service: use the Gmail or Office 365 tutorials above

## 4. How do you want to store passwords?

- I want the safest setup: use [How to Use a Password Manager](/home/mutt/rtd/docs/docs/howto/password-manager.md)
- I do not mind typing passwords when prompted: you can skip this for now

## 5. What should I read next?

Once NeoMutt starts and you can open a mailbox, here are good next steps:

- Learn the basics of the interface: [Getting Started](/home/mutt/rtd/docs/docs/tutorials/getting-started.md)
- Learn to send a message: [Sending Your First Email](/home/mutt/rtd/docs/docs/tutorials/first-email.md)
- Learn to search quickly: [Searching and Filtering Email](/home/mutt/rtd/docs/docs/tutorials/searching-email.md)
