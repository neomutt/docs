The following are the commands understood by NeoMutt.

| Command               | Syntax                                                                         | Description                                                                                                                                                                                                                                                                                                                                                                                             |
|-----------------------|--------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `account-hook`        | `account-hook regex command`                                                   | Execute command when NeoMutt needs to access a matching account                                                                                                                                                                                                                                                                                                                                         |
| `alias`               | `alias [-group name ...] key address [, address ...]`                          | Define an email alias                                                                                                                                                                                                                                                                                                                                                                                   |
| `unalias`             | `unalias {* \| key ...}`                                                       | Remove an email alias                                                                                                                                                                                                                                                                                                                                                                                   |
| `alternates`          | `alternates [-group name ...] regex [regex ...]`                               | Define alternate email addresses belonging to the user                                                                                                                                                                                                                                                                                                                                                  |
| `unalternates`        | `unalternates {* \| regex ...}`                                                | Remove alternate address patterns                                                                                                                                                                                                                                                                                                                                                                       |
| `alternative-order`   | `alternative-order mime-type[/subtype] [...]`                                  | Set preferred order of MIME alternatives                                                                                                                                                                                                                                                                                                                                                                |
| `alternative_order`   | {bdg-danger-line}`Deprecated`                                                  | Use `alternative-order` instead                                                                                                                                                                                                                                                                                                                                                                         |
| `unalternative-order` | `unalternative-order {* \| mime-type[/subtype] ...}`                           | Remove alternative order entries                                                                                                                                                                                                                                                                                                                                                                        |
| `unalternative_order` | {bdg-danger-line}`Deprecated`                                                  | Use `unalternative-order` instead                                                                                                                                                                                                                                                                                                                                                                       |
| `attachments`         | `attachments {+ \| -} disposition mime-type [...]`                             | Configure attachment counting                                                                                                                                                                                                                                                                                                                                                                           |
| `unattachments`       | `unattachments {+ \| -} disposition mime-type [...]`                           | Remove attachment counting rules                                                                                                                                                                                                                                                                                                                                                                        |
| `attachments`         | `attachments ?`                                                                | Query current attachment settings                                                                                                                                                                                                                                                                                                                                                                       |
| `unattachments`       | `unattachments *`                                                              | Remove all attachment rules                                                                                                                                                                                                                                                                                                                                                                             |
| `auto-view`           | `auto-view mime-type[/subtype] [...]`                                          | Automatically view MIME types using mailcap                                                                                                                                                                                                                                                                                                                                                             |
| `auto_view`           | {bdg-danger-line}`Deprecated`                                                  | Use `auto-view` instead                                                                                                                                                                                                                                                                                                                                                                                 |
| `unauto-view`         | `unauto-view {* \| mime-type[/subtype] ...}`                                   | Remove auto-view entries                                                                                                                                                                                                                                                                                                                                                                                |
| `unauto_view`         | {bdg-danger-line}`Deprecated`                                                  | Use `unauto-view` instead                                                                                                                                                                                                                                                                                                                                                                               |
| `bind`                | `bind map[,map ...] key function`                                              | Bind a key to a function in a map                                                                                                                                                                                                                                                                                                                                                                       |
| `unbind`              | `unbind {* \| map[,map ...]} [key]`                                            | Remove a key binding                                                                                                                                                                                                                                                                                                                                                                                    |
| `cd`                  | `cd [directory]`                                                               | Change NeoMutt's working directory; relative paths in subsequent `source`, `change-folder`, etc. are resolved against the new directory. Omitting *directory* resets to `$HOME`.                                                                                                                                                                                                                        |
| `charset-hook`        | `charset-hook alias charset`                                                   | Define a charset alias                                                                                                                                                                                                                                                                                                                                                                                  |
| `iconv-hook`          | `iconv-hook charset local-charset`                                             | Define a charset conversion                                                                                                                                                                                                                                                                                                                                                                             |
| `color`               | `color object [attribute ...] foreground background`                           | Set colour for an object                                                                                                                                                                                                                                                                                                                                                                                |
| `color`               | `color {header \| body} [attr ...] fg bg regex`                                | Set colour for header/body regex match                                                                                                                                                                                                                                                                                                                                                                  |
| `color`               | `color index [attr ...] fg bg [pattern]`                                       | Set colour for index pattern match                                                                                                                                                                                                                                                                                                                                                                      |
| `uncolor`             | `uncolor {index \| header \| body} {* \| pattern ...}`                         | Remove colour rules                                                                                                                                                                                                                                                                                                                                                                                     |
| `crypt-hook`          | `crypt-hook regex keyid`                                                       | Associate a key with addresses matching regex                                                                                                                                                                                                                                                                                                                                                           |
| `echo`                | `echo message`                                                                 | Print a message to the message window                                                                                                                                                                                                                                                                                                                                                                   |
| `exec`                | `exec function [function ...]`                                                 | Execute one or more functions                                                                                                                                                                                                                                                                                                                                                                           |
| `fcc-save-hook`       | `fcc-save-hook pattern mailbox`                                                | Set default Fcc and save folder                                                                                                                                                                                                                                                                                                                                                                         |
| `fcc-hook`            | `fcc-hook pattern mailbox`                                                     | Set default Fcc folder                                                                                                                                                                                                                                                                                                                                                                                  |
| `save-hook`           | `save-hook pattern mailbox`                                                    | Set default save folder                                                                                                                                                                                                                                                                                                                                                                                 |
| `folder-hook`         | `folder-hook [-noregex] regex command`                                         | Execute command when entering a folder                                                                                                                                                                                                                                                                                                                                                                  |
| `group`               | `group [-group name ...] {-rx regex ... \| -addr address ...}`                 | Add addresses/patterns to a group                                                                                                                                                                                                                                                                                                                                                                       |
| `ungroup`             | `ungroup [-group name ...] {* \| -rx regex ... \| -addr address ...}`          | Remove addresses/patterns from a group                                                                                                                                                                                                                                                                                                                                                                  |
| `header-order`        | `header-order header [header ...]`                                             | Set order of displayed headers                                                                                                                                                                                                                                                                                                                                                                          |
| `hdr_order`           | {bdg-danger-line}`Deprecated`                                                  | Use `header-order` instead                                                                                                                                                                                                                                                                                                                                                                              |
| `unheader-order`      | `unheader-order {* \| header ...}`                                             | Remove header order entries                                                                                                                                                                                                                                                                                                                                                                             |
| `unhdr_order`         | {bdg-danger-line}`Deprecated`                                                  | Use `unheader-order` instead                                                                                                                                                                                                                                                                                                                                                                            |
| `hooks`               | `hooks`                                                                        | List all defined hooks                                                                                                                                                                                                                                                                                                                                                                                  |
| `ifdef`               | `ifdef symbol "config-command [args...]"`                                      | Execute command if symbol is defined                                                                                                                                                                                                                                                                                                                                                                    |
| `ifndef`              | `ifndef symbol "config-command [args...]"`                                     | Execute command if symbol is not defined                                                                                                                                                                                                                                                                                                                                                                |
| `finish`              | `finish`                                                                       | Stop processing the current config file                                                                                                                                                                                                                                                                                                                                                                 |
| `ignore`              | `ignore string [string ...]`                                                   | Ignore message headers matching string                                                                                                                                                                                                                                                                                                                                                                  |
| `unignore`            | `unignore {* \| string ...}`                                                   | Stop ignoring headers                                                                                                                                                                                                                                                                                                                                                                                   |
| `index-format-hook`   | `index-format-hook name [!]pattern format-string`                              | Define conditional index format strings                                                                                                                                                                                                                                                                                                                                                                 |
| `lists`               | `lists [-group name ...] regex [regex ...]`                                    | Define mailing list addresses                                                                                                                                                                                                                                                                                                                                                                           |
| `unlists`             | `unlists {* \| regex ...}`                                                     | Remove mailing list definitions                                                                                                                                                                                                                                                                                                                                                                         |
| `subscribe`           | `subscribe [-group name ...] regex [regex ...]`                                | Mark mailing lists as subscribed                                                                                                                                                                                                                                                                                                                                                                        |
| `unsubscribe`         | `unsubscribe {* \| regex ...}`                                                 | Unsubscribe from mailing lists                                                                                                                                                                                                                                                                                                                                                                          |
| `lua`                 | `lua lua-command`                                                              | Execute a Lua command                                                                                                                                                                                                                                                                                                                                                                                   |
| `lua-source`          | `lua-source filename`                                                          | Source a Lua script file                                                                                                                                                                                                                                                                                                                                                                                |
| `macro`               | `macro map[,map ...] key sequence [description]`                               | Define a keyboard macro                                                                                                                                                                                                                                                                                                                                                                                 |
| `unmacro`             | `unmacro {* \| map[,map ...]} [key]`                                           | Remove a keyboard macro                                                                                                                                                                                                                                                                                                                                                                                 |
| `mailboxes`           | `mailboxes [-label label] [-notify\|-nonotify] [-poll\|-nopoll] mailbox [...]` | Define mailboxes to monitor for new mail                                                                                                                                                                                                                                                                                                                                                                |
| `named-mailboxes`     | `named-mailboxes [-notify\|-nonotify] [-poll\|-nopoll] label mailbox [...]`    | Define named mailboxes                                                                                                                                                                                                                                                                                                                                                                                  |
| `unmailboxes`         | `unmailboxes {* \| mailbox ...}`                                               | Remove mailbox definitions                                                                                                                                                                                                                                                                                                                                                                              |
| `mailto-allow`        | `mailto-allow {* \| header-field ...}`                                         | Allow header fields in mailto: URLs                                                                                                                                                                                                                                                                                                                                                                     |
| `mailto_allow`        | {bdg-danger-line}`Deprecated`                                                  | Use `mailto-allow` instead                                                                                                                                                                                                                                                                                                                                                                              |
| `unmailto-allow`      | `unmailto-allow {* \| header-field ...}`                                       | Disallow header fields in mailto: URLs                                                                                                                                                                                                                                                                                                                                                                  |
| `unmailto_allow`      | {bdg-danger-line}`Deprecated`                                                  | Use `unmailto-allow` instead                                                                                                                                                                                                                                                                                                                                                                            |
| `mbox-hook`           | `mbox-hook [-noregex] regex mailbox`                                           | Move read messages to a mailbox                                                                                                                                                                                                                                                                                                                                                                         |
| `message-hook`        | `message-hook pattern command`                                                 | Execute command when reading a matching message                                                                                                                                                                                                                                                                                                                                                         |
| `mime-lookup`         | `mime-lookup mime-type[/subtype] [...]`                                        | Look up MIME type by file extension                                                                                                                                                                                                                                                                                                                                                                     |
| `mime_lookup`         | {bdg-danger-line}`Deprecated`                                                  | Use `mime-lookup` instead                                                                                                                                                                                                                                                                                                                                                                               |
| `unmime-lookup`       | `unmime-lookup {* \| mime-type[/subtype] ...}`                                 | Remove MIME lookup entries                                                                                                                                                                                                                                                                                                                                                                              |
| `unmime_lookup`       | {bdg-danger-line}`Deprecated`                                                  | Use `unmime-lookup` instead                                                                                                                                                                                                                                                                                                                                                                             |
| `mono`                | `mono object attribute`                                                        | Set monochrome attribute for an object                                                                                                                                                                                                                                                                                                                                                                  |
| `mono`                | `mono {header \| body} attribute regex`                                        | Set monochrome attribute for header/body match                                                                                                                                                                                                                                                                                                                                                          |
| `mono`                | `mono index-object attribute pattern`                                          | Set monochrome attribute for index match                                                                                                                                                                                                                                                                                                                                                                |
| `unmono`              | `unmono {index-object \| header \| body} {* \| pattern ...}`                   | Remove monochrome rules                                                                                                                                                                                                                                                                                                                                                                                 |
| `my-header`           | `my-header string`                                                             | Add a custom header to outgoing messages                                                                                                                                                                                                                                                                                                                                                                |
| `my_hdr`              | {bdg-danger-line}`Deprecated`                                                  | Use `my-header` instead                                                                                                                                                                                                                                                                                                                                                                                 |
| `unmy-header`         | `unmy-header {* \| field ...}`                                                 | Remove custom headers                                                                                                                                                                                                                                                                                                                                                                                   |
| `unmy_hdr`            | {bdg-danger-line}`Deprecated`                                                  | Use `unmy-header` instead                                                                                                                                                                                                                                                                                                                                                                               |
| `open-hook`           | `open-hook regex "shell-command"`                                              | Define command to open a compressed mailbox                                                                                                                                                                                                                                                                                                                                                             |
| `close-hook`          | `close-hook regex "shell-command"`                                             | Define command to close a compressed mailbox                                                                                                                                                                                                                                                                                                                                                            |
| `append-hook`         | `append-hook regex "shell-command"`                                            | Define command to append to a compressed mailbox                                                                                                                                                                                                                                                                                                                                                        |
| `push`                | `push string`                                                                  | Push a string of keystrokes onto the **beginning** of the input buffer (contrast with `macro`, which appends to the end). *string* may contain control characters (`\Cx`), key names (`<Return>`), and function names (`<next-page>`).                                                                                                                                                                  |
| `reply-hook`          | `reply-hook pattern command`                                                   | Execute command when replying to a matching message                                                                                                                                                                                                                                                                                                                                                     |
| `send-hook`           | `send-hook pattern command`                                                    | Execute command when sending a matching message                                                                                                                                                                                                                                                                                                                                                         |
| `send2-hook`          | `send2-hook pattern command`                                                   | Execute command when changing a message during composition                                                                                                                                                                                                                                                                                                                                              |
| `score`               | `score pattern value`                                                          | Assign a score to messages matching pattern                                                                                                                                                                                                                                                                                                                                                             |
| `unscore`             | `unscore {* \| pattern ...}`                                                   | Remove score entries                                                                                                                                                                                                                                                                                                                                                                                    |
| `set`                 | `set [no\|inv\|&]variable[?] [...]`                                            | Set, query, toggle, or reset a config option                                                                                                                                                                                                                                                                                                                                                            |
| `set`                 | `set variable=value [variable+=increment] [variable-=decrement] [...]`         | Assign a value to a config option                                                                                                                                                                                                                                                                                                                                                                       |
| `unset`               | `unset variable [variable ...]`                                                | Unset a boolean config option                                                                                                                                                                                                                                                                                                                                                                           |
| `reset`               | `reset variable [variable ...]`                                                | Reset a config option to its default                                                                                                                                                                                                                                                                                                                                                                    |
| `toggle`              | `toggle variable [variable ...]`                                               | Toggle a boolean config option                                                                                                                                                                                                                                                                                                                                                                          |
| `setenv`              | `setenv {variable? \| variable=value}`                                         | Set or query an environment variable                                                                                                                                                                                                                                                                                                                                                                    |
| `unsetenv`            | `unsetenv variable`                                                            | Remove an environment variable                                                                                                                                                                                                                                                                                                                                                                          |
| `sidebar-pin`         | `sidebar-pin mailbox [mailbox ...]`                                            | Pin a mailbox to the top of the sidebar                                                                                                                                                                                                                                                                                                                                                                 |
| `sidebar_pin`         | {bdg-danger-line}`Deprecated`                                                  | Use `sidebar-pin` instead                                                                                                                                                                                                                                                                                                                                                                               |
| `sidebar-unpin`       | `sidebar-unpin {* \| mailbox ...}`                                             | Unpin a mailbox from the sidebar                                                                                                                                                                                                                                                                                                                                                                        |
| `sidebar_unpin`       | {bdg-danger-line}`Deprecated`                                                  | Use `sidebar-unpin` instead                                                                                                                                                                                                                                                                                                                                                                             |
| `source`              | `source filename [filename ...]`                                               | Read configuration commands from a file. | `spam`                | `spam regex [format]`                                                          | Define a spam detection pattern                                                                                                                                                                                                                                                                                                                                                                         |

| `timeout-hook`        | `timeout-hook command`                                                         | Execute command on timeout                                                                                                                                                                                                                                                                                                                                                                              |
| `startup-hook`        | `startup-hook command`                                                         | Execute command at startup                                                                                                                                                                                                                                                                                                                                                                              |
| `shutdown-hook`       | `shutdown-hook command`                                                        | Execute command at shutdown                                                                                                                                                                                                                                                                                                                                                                             |
| `unhook`              | `unhook {* \| hook-type}`                                                      | Remove hooks                                                                                                                                                                                                                                                                                                                                                                                            |
| `version`             | `version`                                                                      | Display NeoMutt version information                                                                                                                                                                                                                                                                                                                                                                     |
