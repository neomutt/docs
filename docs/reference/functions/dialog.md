# Dialog Menu

A simple informational pop-up, such as the help screen or log messages.
It only supports quitting.

| Function | Default Keys | Description                      |
|----------|--------------|----------------------------------|
| `<exit>` |              | Exit this menu                   |
| `<quit>` | `q`          | Save changes to mailbox and quit |
