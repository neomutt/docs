# Simple Pagers

The Simple Pager displays read-only text output that you can scroll through page by page.

Several NeoMutt features produce text output that doesn't fit on a single screen — such as configuration dumps, key binding listings, help text, and version information.
Rather than printing this to the terminal and losing it, NeoMutt displays it in a Simple Pager where you can scroll up and down, search for text, and read at your own pace.
The Simple Pager is also used to display the output of external commands, attachment viewers, and log messages.
